## eB-Mini

--- 
eB-Mini is web application that provides simple internet bank functionality. 

--- 

### Requirements
php: >=7.4.0 
<br> 
MySQL 

--- 
### Installation 
After cloning project
Run:

    composer install
Run:

    npm install


Change .env.example file name to .env
<br>
In .env file configure application name and database connection.
